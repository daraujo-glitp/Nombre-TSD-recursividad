# Taller de Recursividad en Java

## Autor
Nombre: Dilant Yusef Araujo Mosquera  
Programa: Tecnologia en Desarrollo de Software  

## Descripcion
Este proyecto contiene una serie de ejercicios desarrollados en Java utilizando recursion. 
Cada ejercicio se encuentra implementado en una clase independiente, siguiendo una organizacion clara dentro del paquete del proyecto.

## Contenido
El proyecto incluye los siguientes ejercicios:

- Ejercicio 1: Calculo de factorial
- Ejercicio 2: Invertir un numero
- Ejercicio 3: Sumatoria 1 + 1/2 + 1/3 + ... + 1/n
- Ejercicio 4: Suma de digitos de un numero
- Ejercicio 5: Sumatoria de numeros hasta n
- Ejercicio 6: Calculo de potencia
- Ejercicio 7: Maximo comun divisor (MCD)
- Ejercicio 8: Copia de una cadena
- Ejercicio 9: Division por restas sucesivas
- Ejercicio 10: Multiplicacion por sumas sucesivas
- Ejercicio 11: Suma de elementos de un vector
- Ejercicio 12: Suma de elementos de una matriz
- Ejercicio 13: Serie de Fibonacci
- Ejercicio 14: Funcion de Ackermann

## Ejecucion
Para ejecutar el proyecto:

1. Abrir el proyecto en NetBeans
2. Ubicar el paquete EjerciciosDeRecursividad
3. Ejecutar la clase del ejercicio que se desea probar
4. Ingresar los datos solicitados en la consola

## Notas
- Todos los ejercicios fueron desarrollados utilizando recursion
- Cada problema esta resuelto de manera independiente
- Se incluyen validaciones basicas en algunos casos
